# Finora: AI-Powered Personal Finance & Wealth Management Application

Finora is a comprehensive, full-stack personal finance application designed to help users take total control of their financial health. Built with a mobile-first approach, the application bridges the gap between traditional budgeting tools and next-generation artificial intelligence by integrating financial tracking with a dynamic, custom-trained Neural Coach.

## Architecture

The application is structured as a powerful two-part framework:
- **Frontend**: Built using React Native and the Expo Router framework, delivering a flawlessly smooth, cross-platform mobile experience for both iOS and Android. The user interface features premium, glassmorphic design principles, vibrant dynamic gradients, and natively tailored responsive layouts that prioritize user experience.
- **Backend API**: The backend infrastructure is driven by a robust Python Django REST Framework API. It utilizes secure, stateful JWT (JSON Web Token) authentication to manage persistent user sessions across multiple device interactions.

## Key Features

- 💸 **Intelligent Expense Tracking**: Users can easily categorize, log, and monitor their daily expenses with an intuitive dynamic form system.
- 🎯 **Financial Goal Architect**: A robust goal-tracking ecosystem allows users to set, edit, and visualize their savings targets for vacations, emergency funds, and major purchases via real-time progress indicators.
- 📈 **Investment Management**: A dedicated ledger to track asset growth and capital allocations.
- 🧠 **The Neural Coach (AI)**: Chat natively in real-time with an integrated artificial intelligence module. The Neural Coach helps assess financial data, calculate spending trajectories, and deliver customized wealth-optimization strategies using a pristine, identical-to-native messaging UI.

## Local Implementation

This codebase natively supports robust data isolation, environment configuration protection via `.gitignore`, and is engineered for scalability. Finora isn't just a spreadsheet on your phone; it is a proactive, data-driven financial assistant meant to elevate modern wealth management.
